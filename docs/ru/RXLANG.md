# RXLang

Язык страниц в mesh. Не HTML, не JS. Компилятор — в полном образе.

```rxsrc
@title Витрина
@section Услуги
@text Хостинг в mesh
@wallet btc:bc1q...
@link rxtx://... заказ
@escrow lst-001
```

| | |
|---|---|
| `@title` | Заголовок |
| `@section` | Раздел |
| `@text` | Текст |
| `@link` | `rxtx://` |
| `@wallet` | Кошелёк |
| `@escrow` | Escrow |

**Сообщество → Конструктор.** Нужен слот хостинга в **Вход RXTX**.
