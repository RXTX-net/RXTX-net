# RXTX-NET

Любое твоё железо (Pi, мини-ПК, x86) на Debian 12: роутер, VPN/Tor, Workstation в Docker, mesh, RXLang. Сборка под `arm` / `arm64` / `amd64`.

**Shell** — оплата и установка (одна прошивка).  
**Полный RXTX** — панель и всё остальное. Ставится **на месте** после Matrixarch.

```
shell.img → оплата → «Установить» → WSSIL → reboot → rxtx.local
```

| Контур | Что делает |
|--------|------------|
| Хост | Панель, VPN, mesh, firewall |
| Docker | Браузер и терминал для clearnet |
| Mesh | Чат, страницы RXLang, рынок |

Полный код не в git. Только shell и эта документация.
