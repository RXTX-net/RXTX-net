# RXTX-NET

Any hardware (Pi, mini PC, x86) on Debian 12: router, VPN/Tor, Docker Workstation, mesh, RXLang. Built for `arm` / `arm64` / `amd64`.

**Shell** — payment and install (single flash).  
**Full RXTX** — panel and everything else. Installed **in place** after Matrixarch.

```
shell.img → payment → Install → WSSIL → reboot → rxtx.local
```

| Layer | What it does |
|-------|----------------|
| Host | Panel, VPN, mesh, firewall |
| Docker | Browser and terminal for clearnet |
| Mesh | Chat, RXLang pages, market |

Full code is not in git. Only shell and this documentation.
