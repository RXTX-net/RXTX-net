# RXLang

Page language for mesh. Not HTML, not JS. Compiler is in the full image.

```rxsrc
@title Storefront
@section Services
@text Mesh hosting
@wallet btc:bc1q...
@link rxtx://... order
@escrow lst-001
```

| | |
|---|---|
| `@title` | Title |
| `@section` | Section |
| `@text` | Text |
| `@link` | `rxtx://` |
| `@wallet` | Wallet |
| `@escrow` | Escrow |

**Community → Constructor.** Hosting slot required in **RXTX Account**.
